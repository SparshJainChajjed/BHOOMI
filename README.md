# BHOOMI

BHOOMI is a simple soil monitoring project. The Arduino sketch reads a pH sensor and prints nutrient advice for a selected crop.

## Hardware
- Arduino-compatible board
- pH sensor connected to analog pin A0

## Getting Started
1. Open `BHOOMI.ino` in the Arduino IDE.
2. Change the `cropType` variable at the top of the sketch if you use a different crop.
3. Upload the sketch to your board.
4. Open the Serial Monitor to see pH readings and recommendations.

Extend the recommendation logic or add sensors as needed for your farm.
